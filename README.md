***Smart Study Planner***
This project is a simple yet effective smart study planning system that helps students automatically generate a study schedule based on exam dates and also

visualizes study distribution with the help of graphs. 

**Overview**
This smart study planner helps students to organise thier study time effectively by taking subjects and thier exam dates as input and creating a structured daily 

time table and also shows how study time is distributed across subjects. 

**Features** 
This project : 
*generates a structured daily time table 
*gives priority to subjects based on exam dates 
*visualizes study distribution using bar graph 
*displays output in table form 

**Technologies used**
*python language 
*datetime module 
*matplotlib library  

**How it works ?**
1.User enters subjects and exam dates 
2.the program sorts subjects based on exam dates , allocates study time accordingly and generates a structured daily study schedule. 
3.a graph shows how study time is distributed 
4.output :  a study plan in table format and a bar graph showing subject wise study time distribution 

**How to run**  
1.Install Python 
2.install required library 
3.run the program 
4.enter the input asked by the program in the required format
